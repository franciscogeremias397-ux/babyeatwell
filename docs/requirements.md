# Requirements

## Goal

Build and maintain the WeChat Mini Program for `宝贝乖乖吃饭`.

## Functional Requirements

- Provide a recipe module with baby profile, weekly meal plan, random meal, and recipe detail pages.
- Provide a Dogdan supervised eating module with call, connected video, interaction buttons, and ending flow.
- Keep recipe features and Dogdan supervised eating features independent.
- Use CloudBase public storage URLs for video and audio playback assets.

## Non-Functional Requirements

- Do not commit secrets or environment files.
- Keep WeChat Mini Program project structure intact.
- Avoid local-only dependencies for playback media.
